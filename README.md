Fitur yang dibuat diantaranya:
✔ Fitur Login untuk masuk ke aplikasi
✔ Fitur Register untuk membuat akun baru
✔ Fitur Home untuk menampilkan gambar postingan berupa List
✔ Fitur Detail Postingan untuk menampilkan detail dari postingan
✔ Fitur Kolom komentar untuk menuliskan komentar ke postingan
✔ Fitur Create Post untuk membuat postingan baru
✔ Fitur Profil untuk menampilkan profil pengguna yang berisi informasi username, nama, info gender, bio, jumlah posts, dan foto profil
✔ Fitur Edit Profil untuk mengedit profil pengguna seperti mengganti foto profil, mengubah username dan nama, serta menambahkan bio
✔ Fitur Edit Postingan untuk mengubah judul postingan dan caption
✔ Fitur Logout untuk keluar dari aplikasi
✔ Notifikasi ketika membuat postingan baru, memperbarui postingan, menghapus postingan.

Cara menjalankan aplikasi:
1.	Login ke aplikasi menggunakan email/password atau menggunakan akun google
2.	Jika ingin membuat akun baru, silahkan daftar terlebih dahulu
3.	Setelah login, Anda akan masuk ke halaman Beranda
4.	Gulir (scroll) layar ke bawah untuk melihat foto-foto yang telah diposting
5.	Ketuk ikon Hati (Heart) pada postingan yang Anda sukai dan jumlah likes akan terlihat
6.	Ketuk pada gambar postingan untuk melihat caption lengkap dan komentar
7.	Terdapat menu option untuk mengedit postingan/menghapus
8.	Klik menu Buat Postingan untuk membuat postingan baru
9.	Paste link gambar yang ingin diposting, akan muncul previewnya. Berikan judul untuk postingan dan caption
10.	Jika sudah, akan ada notifikasi dan postingan akan muncul di bagian Homepage/Feed serta jumlah postingan di Profil juga akan bertambah
11.	Di bagian menu Profil, jika ingin mengubah username, silahkan klik Edit Profil.
12.	Anda bisa mengubah foto profil dengan mempaste link foto yang ingin di jadikan sebagai foto profil, mengubah username, mengubah nama, dan menuliskan bio.
13.	Jika sudah, akan ada notifikasi dan tampilan profil akan berubah
14.	Tekan Logout jika ingin keluar dari aplikasi
15.	Maka akan diarahkan kembali ke tampilan login
